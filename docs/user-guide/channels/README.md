# Channels

Channels are where conversations happen in Chatalot. They live inside [Groups](../groups/README.md), which themselves live inside [Communities](../communities/README.md).

## Channel Types

Chatalot supports two types of channels:

| Type | Description |
|------|-------------|
| **Text** | Standard messaging channels with full support for formatting, reactions, threads, pins, and file sharing |
| **Voice** | Real-time audio and video channels using WebRTC |

## Quick Reference

- Channels are created inside Groups by Group owners and admins
- Channel names can be up to 64 characters
- Channel topics (descriptions) can be up to 512 characters
- Text channels support Markdown formatting, file uploads, reactions, threads, and pins
- Voice channels support up to 25 concurrent participants
- Direct Messages are special channels between two users, existing outside the Group hierarchy
