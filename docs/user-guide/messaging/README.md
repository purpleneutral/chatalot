## Quick Reference

- Press **Enter** to send (configurable to Ctrl+Enter in settings)
- Press **Shift+Enter** for a new line
- Use `@username` to mention a user
- Use Markdown for formatting (bold, italic, code, etc.)
- Right-click a message for the context menu (edit, delete, pin, bookmark, reply, react)
- Messages are encrypted before leaving your device
