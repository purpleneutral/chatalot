# Keyboard Shortcuts

All keyboard shortcuts available in Chatalot, organized by context.

> **Status: Complete** -- Press `?` anywhere in the chat view to open the shortcuts modal.

---

## Messages

These shortcuts work when the message input is focused.

| Shortcut | Action |
|----------|--------|
| `Enter` | Send message (default) |
| `Shift+Enter` | New line (default) |
| `Ctrl+Enter` | Send message (alternative, configurable in Settings > Chat) |
| `Enter` | New line (when send behavior is set to Ctrl+Enter) |
| `Arrow Up` | Edit your last message (when input is empty) |

> **Note:** The send behavior is configurable. Go to **Settings > Chat** to switch between "Enter to send" and "Ctrl+Enter to send".

## Formatting

These shortcuts work when the message input is focused and apply formatting to selected text or insert formatting markers.

| Shortcut | Action |
|----------|--------|
| `Ctrl+B` | **Bold** (wraps selection in `**`) |
| `Ctrl+I` | *Italic* (wraps selection in `_`) |
| `Ctrl+E` | `Inline code` (wraps selection in `` ` ``) |

On macOS, use `Cmd` instead of `Ctrl`.

## Navigation

These shortcuts work globally in the chat view.

| Shortcut | Action |
|----------|--------|
| `Ctrl+K` | Open quick channel/DM switcher |
| `Ctrl+F` | Search messages in current channel |
| `Ctrl+T` | Focus the message input |
| `End` | Jump to latest message (scroll to bottom) |
| `Home` | Scroll to top of messages |
| `Shift+Esc` | Mark all channels as read |
| `Esc` | Close the currently open modal or panel |
| `?` | Show keyboard shortcuts modal |
| `Ctrl+/` | Show keyboard shortcuts modal (alternative) |

On macOS, use `Cmd` instead of `Ctrl`.

## Lightbox

These shortcuts work when the image lightbox is open.

| Shortcut | Action |
|----------|--------|
| `Arrow Left` | Previous image |
| `Arrow Right` | Next image |
| `Esc` | Close lightbox |

## Autocomplete

These shortcuts work when the mention (`@`) or emoji (`:`) autocomplete popup is visible.

| Shortcut | Action |
|----------|--------|
| `Arrow Up` | Select previous suggestion |
| `Arrow Down` | Select next suggestion |
| `Tab` | Accept selected suggestion |
| `Enter` | Accept selected suggestion |
| `Esc` | Dismiss autocomplete |

## Quick Switcher

These shortcuts work when the quick switcher (`Ctrl+K`) is open.

| Shortcut | Action |
|----------|--------|
| `Arrow Up` | Select previous result |
| `Arrow Down` | Select next result |
| `Enter` | Jump to selected channel or DM |
| `Esc` | Close quick switcher |

## Other

| Shortcut | Action |
|----------|--------|
| `Ctrl+V` | Paste image from clipboard as file upload |

## Escape Key Priority

When pressing `Esc`, panels and modals close in this order (whichever is open first):

1. Quick switcher
2. Edit history viewer
3. Image lightbox
4. GIF picker
5. Keyboard shortcuts modal
6. Message search
7. Pinned messages panel
8. Poll panel
9. Create poll form
10. Member panel
11. Thread panel
12. Bookmarks panel
13. Scheduled messages panel
14. Schedule picker
